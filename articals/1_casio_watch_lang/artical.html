With all this info, I have the API to get something showing up on the watches screen! Now I have to go and put togeather a simulator.


I started off by getting my basic projected laid out, just a build.sh file, and a pc_main.c. I really enjoy seeing the 7 segment displays on this watch, so its the first thing that I wanted to see getting drawn. I started by implimenting a simple 7 segment display on my pc.

The simulated LCD displays are all showing a single uint8_t. To make my life a little better and to make refrencing a spesfic Segment easer, I made this enum. 


            /* casio_watch_lang/pc_main.c */
            typedef enum Seg
            {
                /*     a
                 *     _
                 *  f |_|  b  <-- g in the center
                 *  e |_|  c
                 *     d
                 * */
            
                seg_a = 1 << 0,
                seg_b = 1 << 1,
                seg_c = 1 << 2,
                seg_d = 1 << 3,
                seg_e = 1 << 4,
                seg_f = 1 << 5,
                seg_g = 1 << 6,
            } Seg;

For those who dont know, the funny arrows are the bitshift opperator, so this makes it easer for me to say the first bit of the number, or the 2'nd bit of a number. It is equivlent to saying something like this 


            seg_a = 0b00000001
            seg_b = 0b00000010
            seg_c = 0b00000100
            and so on...

lastly, I need to store these numbers somewhere to display, so I put a simple structure togeather to make interacting with the 8 diffrent lcds on this watch. 

            struct LcdState
            {
                // 0 - 5 : main time display, 0,1 hr 2,3min 4,5 sec
                // 6 - 7 : day displ,
            
                uint8_t d[8];
            } lcd_state;
notice how there is no typedef before this struct, so this lcd_state is a global varable to the simulator program.
Next up, we will need a way to turn segments into numbers, and this is done by just "OR-ing" the correct segments togeather to get the right segments on as we want them. I did this for all letters, base10 numbers, and base16 numbers 


    void Set7Seg_Number(uint8_t *v, uint8_t n)
    {
        switch (n)
        {
            case 0: *v = (seg_a | seg_b | seg_c | seg_d | seg_e | seg_f); break;
            case 1: *v = (seg_b | seg_c); break;
            case 2: *v = (seg_a | seg_b | seg_d | seg_g | seg_e); break;
            case 3: *v = (seg_a | seg_b | seg_g | seg_c | seg_d); break;
